Here are some basics.

Interiors of houses are collections of rooms. You construct simple rooms with the room function. You combine simple rooms into more complex rooms with the room-above room-beside functions.

procedure

(room width height floor-texture) → room?

  width : number?
  height : number?
  floor-texture : image?

Makes a new room. Note that the width and height are in tiles, not pixels. Tiles are 32x32 pixels.

Example:

(room 1 1 wood-tile)

Or more interestingly:

(room 3 3 wood-tile)

procedure

(room-beside room ...) → room?

  room : room?

Makes a new room with the supplied rooms placed beside each other. Any overlapping walls will be removed.

(define r (room 3 3 wood-tile))
(define door (room 1 1 wood-tile))
(beside-room (r door r))

If you want to control where that door ends up, you can use the nudge-down function.

(define r (room 3 3 wood-tile))
(define door (room 1 1 wood-tile))
(beside-room (r (nudge-down 1 door) r))

procedure

(room-above room ...) → room?

  room : room?

Makes a new room with the supplied rooms placed above each other. Any overlapping walls will be removed.

See above.

procedure

(room-list->entity-list room) → (listof entity?)

  room : 
(or/c room?
      (listof room?))

This is the function you use to convert a room? to something you can pass into start-game, e.g.:

(start-game
  player
  enemy
  (room-list->entity-list r))
